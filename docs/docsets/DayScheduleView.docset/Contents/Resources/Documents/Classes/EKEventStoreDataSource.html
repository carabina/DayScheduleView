EKEventStoreDataSource

open class EKEventStoreDataSource : DayScheduleViewDataSource

Appointment data source that loads appointments from the user’s calendar database using EventKit.

EKEventStoreDataSource can be used as a default data source to display events in the user’s calendar database in the day schedule view as appointments.
